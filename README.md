# Wage Theft Data Challenge

* [Wage and Hour Enforcement Dataset from DOL](https://enfxfr.dol.gov/data_catalog/WHD/whd_whisard_20180814.csv.zip)

* Working Age Population American Community Survey 5 year 2016
